Detailed Description

This class is used to display the LTC alerts and information. It is also used to display the news alerts.

This class is used to create the section on the web page where the LTC alerts and information are displayed. It also creates the section where the news alerts are displayed. Each category of information are shown in rotation every 30 seconds
