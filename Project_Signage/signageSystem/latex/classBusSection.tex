\doxysubsection{Detailed Description}
This class is used to display the bus schedule and bus map on the signage. 

This class is used to create the section on the web page where the bus schedule and bus map are displayed. The bus schedules are shown in rotation every 30 seconds and the positions of the bus in the shedule are shown on the map
